Meaning of hwpplayers

hwpplayers community is founded in the date November 19, 2015 or earlier.I was reading philosophy books and realized that philosophy discipline is another approach that anaylze science theory and concepts. That is why hwp is "hard working philosophers".Player word is coming from a FPS game named Quake.When you start the game and look for your profile you'll see a player name "player1". For a community name it is players as plural. Thus it became hard working philosophers players which represents "playful cleverness" and "hack value".
Happy hacking !

hwpplayers community has a “level system” like in games. We use that system for learning. We start from the basic ideas and try to learn everything about the idea.That's why we use rainbow logo.
